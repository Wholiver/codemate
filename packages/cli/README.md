# @codemate/cli

JSR package for Codemate CLI-facing module exports.

## Publish to JSR

```bash
npx jsr publish --dry-run
```

## Global command for users

For a global `codemate` command, install this npm package:

```bash
npm install -g codemate-agent
codemate --help
```

## Notes

- JSR packages are module-first and rely on `exports`.
- JSR npm-compat tarballs are generated from package exports metadata.
