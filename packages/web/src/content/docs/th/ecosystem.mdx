ชุดโครงการชุมชนที่สร้างขึ้นบน Codemate

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ Codemate ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-codemate](https://github.com/awesome-codemate/awesome-codemate) และ [codemate.cafe](https://codemate.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
