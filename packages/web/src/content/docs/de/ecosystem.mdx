Eine Sammlung von Community-Projekten, die auf Codemate basieren.

:::note
Möchten Sie Ihr Codemate-bezogenes Projekt zu dieser Liste hinzufügen? Senden Sie einen PR.
:::

Sie können sich auch [awesome-codemate](https://github.com/awesome-codemate/awesome-codemate) und [codemate.cafe](https://codemate.cafe) ansehen, eine Community, die das Ökosystem und die Community zusammenfasst.
